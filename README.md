# Conserving Genetic Diversity: A Case Study on *Trollius europaeus* L and the Role of *ex-Situ* Conservation in Seed-Banks
This repository contains rendered versions of the statistical analysis used for the Master thesis submitted by Michael Ott on April 1st 2026 at the University of Regensburg.
The following files are included:
- **env_Tro.pdf**: Environmental Analysis of *T. europaeus*
- **germination_Tro.pdf**: Germination Analysis of *T. europaeus*
- **migseq_Tro.pdf**: SNPs Analysis of *T. europaeus*
- **migseq_SBCS.pdf**: SNPs Analysis of Commercially Propagated Seeds and Seed-Bank Accessions
